How to Extend the Project

Add a New Sidebar Section

  1. Add new object in AppScreen.kt
  2. Add it to drawerStructure
  3. Register route inside AppNavHost

Add a New Screen Version

  1. Create new screen file (e.g., v3)
  2. Add route in AppScreen
  3. Add composable inside AppNavHost

Add New SMS Filtering

  1. Add new function in SmsReaderRepository
  2. Expose it via ViewModel
  3. Call it from respective screen
